# Docコメント規約

## 必須ルール

1. **すべてのモジュール・メソッド・関数にDocコメントを記載すること**
   - public / private を問わず、すべてに記載する
   - クラス、構造体、トレイト、インターフェースなどの型定義も対象とする

2. **Docコメントはすべて日本語で記載すること**
   - 説明文、パラメータ説明、戻り値説明などすべて日本語で記述する
   - 技術用語（例: HTTP, JSON, API）はそのまま英語表記で構わない

3. **編集したファイルに英語のDocコメントがあれば日本語に書き換えること**
   - 自分が編集対象としたファイル内に英語のDocコメントがあれば、適宜日本語に修正する
   - 編集対象外のファイルまで遡って修正する必要はない
